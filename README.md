<h1>SmallDoubling</h1> 
</div> 

<p><strong>SmallDoubling</strong> is a lightweight Minecraft mod for <strong>NeoForge</strong> that enhances gameplay by allowing players to double small flower drops using bone meal.</p>

<h2>Features</h2>
<ul>
  <li>Right-click small flowers with <code>bone meal</code> to get an extra flower drop.</li>
  <li>Works only on blocks tagged as <code>small_flowers</code>.</li>
  <li>Compatible with Minecraft <strong>1.21.1</strong> using NeoForge.</li>
  <li>Client-friendly: all logic runs server-side to prevent duplication issues.</li>
  <li>Lightweight and easy to integrate into existing worlds or servers.</li>
</ul>

<h2>How to Use</h2>
<ol>
  <li>Hold <code>Bone Meal</code> in your hand.</li>
  <li>Right-click on a small flower (e.g., poppy, daisy).</li>
  <li>The flower will drop an extra item immediately.</li>
  <li>If in Creative mode, bone meal is not consumed.</li>
</ol>

<h2>Installation</h2>
<ul>
  <li>Download the mod jar from the <a href="https://github.com/DerErneuerer/SmallDoubling" target="_blank">GitHub repository</a>.</li>
  <li>Place the jar file into your <code>mods</code> folder.</li>
  <li>Launch Minecraft with the NeoForge mod loader.</li>
</ul>
