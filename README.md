<div style="text-align: center;"> <h1>OPAC BetterCommands</h1> </div> <p> <strong>OPAC BetterCommands</strong> is an addon for <a href="https://github.com/thexaero/open-parties-and-claims" target="_blank">OpenPaC</a>, developed for use on the <a href="https://brassworks.opnsoc.org/" target="_blank">Brassworks SMP</a>. </p> <p> It improves and simplifies OpenPaC commands by providing shorter aliases and easier access for players. </p> <ul> <li><code>/party</code> – Redirects to <code>/openpac-parties</code></li> <li><code>/claims</code> – Redirects to <code>/openpac-claims</code></li> <li><code>/pchat</code> – Alias for <code>/opm</code> (OPAC Party Chat): <ul> <li><code>/pchat toggle</code> – Toggles party chat on/off.</li> <li><code>/pchat status</code> – Shows current chat toggle state.</li> <li><code>/pchat &lt;message&gt;</code> – Sends a message directly to your party chat via /opm.</li> </ul> </li> </ul> <p> Designed for <strong>Minecraft 1.21.1</strong> using the <strong>NeoForge</strong> mod loader. </p>
<h2>Features</h2> <ul> <li>Shortened aliases for OpenPaC commands: <code>/party</code> and <code>/claims</code>.</li> <li>Unified party chat control via <code>/pchat</code> (alias for <code>/opm</code>).</li> <li>Lightweight, dependency-aware redirects for seamless integration.</li> <li>Fully open-source and easily extendable.</li> </ul>
